Nut feeder using color detection

Abstract : Usui International Corporation (Thailand) in the production of brake hoses, fuel hoses, and high-pressure injection hoses for diesel and gasoline engines. As part of a cooperative study project, the company was tasked with developing a program to detect object colors within a Part Feeder machine using Python programming language alongside an industrial camera. This tool served to inspect color disparities in components, aiming to reduce errors in the production process. The experiment focused on distinguishing between two colors, silver and black, utilizing Python to detect colors using the RGB model. The results showed increased production efficiency and a decrease in errors. The Python program developed efficiently detected color variations in the components, demonstrating its effectiveness compared to methods that did not utilize the previous industrial camera setup.
